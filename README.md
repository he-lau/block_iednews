# block_iednews

Bloc d’actualités compatible avec Moodle 4.1.

Le plugin permet aux administrateurs du site d’ajouter, modifier, publier,
planifier et supprimer des actualités depuis `/blocks/iednews/manage.php`.
Il prend en charge les images intégrées, le défilement automatique accessible
et les demandes de confidentialité Moodle.

Installation : placer le dossier `iednews` dans `blocks/`, puis lancer la mise
à niveau Moodle depuis l’administration du site ou avec `admin/cli/upgrade.php`.
